The Tradeable Ownership of General Agreement (TOGA) allows Superfluid cashflows to be wrapped as NFTs, making them tradeable and composable with other DeFi protocols. This enables new use cases like cashflow securitization and secondary markets.

## Overview

TOGA provides:
- NFT wrapper for Superfluid streams and distributions
- Tradeable cashflow ownership
- Composability with NFT marketplaces
- Cashflow splitting and bundling

## Read Functions

### getFlowNFTInfo

Returns information about a flow NFT.

<Tabs items={['ABI', 'Actions', 'Hooks']}>
<Tab value="ABI">
```ts
import { toga } from '@superfluid-finance/sdk/abi';

const nftInfo = await client.readContract({
  address: '0x...',
  abi: toga,
  functionName: 'getFlowNFTInfo',
  args: [tokenId],
});
```
</Tab>
<Tab value="Actions">
```ts
import { readToga } from '@superfluid-finance/sdk/actions';

const nftInfo = await readToga(client, {
  address: '0x...',
  functionName: 'getFlowNFTInfo',
  args: [tokenId],
});
```
</Tab>
<Tab value="Hooks">
```ts
import { useReadToga } from '@superfluid-finance/sdk/wagmi';

const { data: nftInfo } = useReadToga({
  address: '0x...',
  functionName: 'getFlowNFTInfo',
  args: [tokenId],
});
```
</Tab>
</Tabs>

### ownerOf

Returns the owner of a cashflow NFT.

### balanceOf

Returns the number of cashflow NFTs owned by an address.

### tokenURI

Returns the metadata URI for a cashflow NFT.

## Write Functions

### wrapFlow

Wraps an existing Superfluid stream as an NFT.

<Tabs items={['ABI', 'Actions', 'Hooks']}>
<Tab value="ABI">
```ts
import { toga } from '@superfluid-finance/sdk/abi';

const hash = await client.writeContract({
  address: '0x...',
  abi: toga,
  functionName: 'wrapFlow',
  args: [token, sender, receiver],
});
```
</Tab>
<Tab value="Actions">
```ts
import { writeToga } from '@superfluid-finance/sdk/actions';

const hash = await writeToga(client, {
  address: '0x...',

```
</Tab>

### unwrapFlow

Unwraps a cashflow NFT back to a regular stream.

### transferFrom

Transfers ownership of a cashflow NFT.

### approve

Approves another address to transfer a specific cashflow NFT.

### setApprovalForAll

Approves an operator to manage all cashflow NFTs.

## Events

- `FlowWrapped` - Emitted when a flow is wrapped as NFT
- `FlowUnwrapped` - Emitted when an NFT is unwrapped
- `Transfer` - Standard ERC721 transfer event
- `Approval` - Standard ERC721 approval event
- `ApprovalForAll` - Standard ERC721 operator approval event
